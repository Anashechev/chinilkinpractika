## 🚀 Подготовка к демонстрации

### 1. Создание утилит шифрования
```python
# tickets/utils.py
from cryptography.fernet import Fernet
from django.conf import settings
import base64

class DataEncryption:
    def __init__(self):
        # Генерация ключа из SECRET_KEY
        key = base64.urlsafe_b64encode(settings.SECRET_KEY.encode()[:32].ljust(32, b'0'))
        self.cipher = Fernet(key)
    
    def encrypt(self, data):
        if isinstance(data, str):
            data = data.encode()
        return self.cipher.encrypt(data)
    
    def decrypt(self, encrypted_data):
        decrypted = self.cipher.decrypt(encrypted_data)
        return decrypted.decode()
```

### Возможные вопросы и ответы:

**Q: Как обеспечивается защита от XSS-атак?**
A: Через Django шаблонизатор (автоэкранирование), валидацию форм, и дополнительную проверку регулярными выражениями.

**Q: Как реализовано шифрование данных?**
A: Используется Fernet (AES-128-CBC) с ключом на основе SECRET_KEY, обеспечивает конфиденциальность и целостность.

**Q: Как мониторится безопасность системы?**
A: Через Django middleware, логирование в SecurityLog модель, автоматическое обнаружение подозрительной активности.

**Q: Как обеспечивается резервное копирование?**
A: Автоматические команды management, сжатие данных gzip, проверка целостности, возможность восстановления.
