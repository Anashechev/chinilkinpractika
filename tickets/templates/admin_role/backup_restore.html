Резервное копирование и восстановление базы данных

На этой странице вы можете создать резервную копию базы данных или восстановить её из ранее созданной копии.

Создать резервную копию

Создайте полную резервную копию текущей базы данных. Файл будет автоматически загружен на ваше устройство.

{% csrf_token %}
Восстановить из резервной копии

Восстановите базу данных из загруженного файла резервной копии.

Внимание: При восстановлении текущая база данных будет заменена. Перед восстановлением старая база данных будет скачана на ваше устройство с именем файла olddb_*.sqlite3.

{% csrf_token %}
Информация о резервном копировании
  • Резервные копии загружаются прямо на ваше устройство
  • Перед восстановлением старая база данных скачивается на ваше устройство с именем файла olddb_*.sqlite3
  • Формат файлов резервных копий: db_backup_YYYYMMDD_HHMMSS.sqlite3
  • При восстановлении используйте только файлы резервных копий, созданные этой системой
